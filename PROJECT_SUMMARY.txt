═══════════════════════════════════════════════════════════════════════════

Nombre del Proyecto: TurnosSaludProg2
Tipo: Aplicación de Escritorio Java
Propósito: Gestión de turnos médicos para centros de salud comunitarios
Arquitectura: Tres Capas (Presentation, Business Logic, Data Access)
Lenguaje: Java 11+
Interfaz: Consola (CLI)

FUNCIONALIDADES IMPLEMENTADAS
═══════════════════════════════════════════════════════════════════════════

✓ Sistema de Autenticación
  - Login con usuario y contraseña
  - Control de acceso basado en roles (RBAC)
  - Gestión de sesiones
  - Usuarios activos/inactivos

✓ Gestión de Personal y Usuarios
  - Registro de médicos con especialidades
  - Registro de personal administrativo
  - Registro de pacientes
  - Asignación de horarios de atención
  - Listado de usuarios por rol

✓ Gestión de Turnos
  - Creación de turnos con validaciones
  - Visualización de turnos por paciente/médico
  - Cancelación de turnos con motivo
  - Confirmación de turnos
  - Completado de turnos con observaciones
  - Estados: Pendiente, Confirmado, Completado, Cancelado

✓ Disponibilidad de Turnos
  - Consulta por especialidad médica
  - Consulta por profesional
  - Generación automática de horarios disponibles
  - Prevención de doble reserva
  - Cálculo en tiempo real

✓ Roles de Usuario
  - ADMINISTRATIVO: Gestión completa del sistema
  - MEDICO: Visualización de agenda y atención de pacientes
  - PACIENTE: Solicitud y gestión de turnos propios
